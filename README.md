# Journey · Amber's Experience Atlas

My career journey as an interactive atlas — six fanned experience cards + a half-dome globe that rotates to the active stop. USC (LA) → CBS (San Diego) → TIKTAL (London) → Meituan → bilibili → Noumena (Shanghai).

**Live**: https://hiamberhuang.github.io/journey/

- Vanilla JS + SVG + CSS, single HTML file, zero dependencies
- Bilingual (EN / 中), click a card to travel, click again to flip
- Designed to be embedded anywhere an iframe goes — Feishu docs included
- Built on my open-source component: [atlas-experience-demo](https://github.com/hiamberhuang/atlas-experience-demo)
